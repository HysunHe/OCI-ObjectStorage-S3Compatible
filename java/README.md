# 使用 AWS Java SDK 操作 OCI 对象存储

## 描述

样例程序演示了如何使用OCI信息配置AWS S3 Java SDK，并且实现了从 Bucket 获取上传和批量删除两个功能。关键是如何利用OCI信息配置AWS S3 SDK，如果需要实现更多的功能，那么直接参考AWS S3 SDK的官方文档中的实例代码即可。
